# telco-churn-analysis
Customer churn analysis using Python and Power BI

## Project Overview

Customer churn is a critical problem for subscription-based businesses. When customers leave a service, it directly impacts revenue and growth.

This project analyzes telecom customer churn data to identify the key drivers of churn and estimate the revenue at risk. The analysis combines **Python data exploration** with an **interactive Power BI dashboard** to provide business insights.

The goal of this project is to demonstrate how data analytics can help companies understand customer behavior and reduce churn.
